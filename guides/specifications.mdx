# Technical Specifications

Detailed technical information for seamless integration.

## Data Format & Structure

### JSON Schema
```json
{
  "instruction": "string",
  "input": "string", 
  "output": "string"
}
```

### Field Descriptions
| Field         | Description                                     |
|---------------|-------------------------------------------------|
| `instruction` | Context or role definition for the AI system  |
| `input`       | Medical question or patient scenario            |
| `output`      | Medical answer, advice, or diagnosis            |
